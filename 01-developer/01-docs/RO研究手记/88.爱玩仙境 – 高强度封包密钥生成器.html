爱玩仙境 – 高强度封包密钥生成器

2015年4月8日2015年4月9日 Sola丶小克 原创工具

这是一款用来生成高强度封包密钥组合的工具，生成出来的密钥绝对可以防止WPE等封包重发器。你也可以提供一组密钥，然后用本工具来测试其强度是否能够防止WPE等封包重发器。

产品名称：爱玩仙境 – 高强度封包密钥生成器

最新版本：1.0.0.1

更新日期：2015年04月08日

作用说明：此软件可以生成能防住WPE等封包重发器的PacketKeys（即封包密钥），也可以测算一组密钥是否能防住WPE封包重发器。

下载地址：本地下载

安装说明

若您使用的是Win7及更新的系统，那么可以直接运行本软件。

若您使用的是WinXP或Win2003系统，那么使用本软件之前，您的电脑需 要先安装 Microsoft .NET Framework 4 Client Profile 否则的话 运行可能会出错。

您可以到微软网站上下载这个 Framework 组件： http://download.microsoft.com/download/5/6/2/562A10F9-C9F4-4313-A044-9C94E0A8FAC8/dotNetFx40_Client_x86_x64.exe

使用说明

1、点击“生成强力密钥”按钮可以生成一组由三个密钥组成的高强度封包加密密钥（绝对能防WPE）

2、你也可以在“暴力测算”中填写三个密钥，然后点击“进行暴力测算”按钮，以此来测试其是否可以防住WPE等封包重发器

3、部分特殊版本的客户端（比如2013-12-23版），可能第二个密钥和第三个密钥要求必须是相同的，此时你可以先勾选“生成第二个和第三个密钥一致的强力密钥”复选框，然后再点击“生成强力密钥”
