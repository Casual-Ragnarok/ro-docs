解决官方发型无法换发色的问题

2015年2月6日2015年3月15日 Sola丶小克 未分类

如果大家根据之前《什么是DIFF客户端以及DIFF工具介绍》所讲解的方法来DIFF客户端，那么可能会发现DIFF完成后无法选择官方发型的发色。这是因为DIFF时我们启用了“多发型支持”选项导致的，这选项叫“Enable 64k Hairstyle”！

其实要解决这个问题也很简单，覆盖一下我提供的一个修正补丁就行了，造成这个问题的原因是，启用了多发型支持后，客户端读取发色文件的方式被改变导致的。这个修正图档是我在国外一个小论坛找到的，忘记出处了……

在应用补丁之前，可以看到新建角色界面，更改发色时角色一点变化都没有：

在应用补丁之后，可以看到恢复正常了，不过发型可能和官服稍微有一些差异：
