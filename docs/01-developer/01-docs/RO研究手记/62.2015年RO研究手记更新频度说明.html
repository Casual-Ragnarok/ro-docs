2015年RO研究手记更新频度说明

2015年3月1日 Sola丶小克 未分类

各位好，首先感谢大家一直以来支持RO研究手记。小克过年期间很忙，和媳妇儿领结婚证，拜访亲戚等。回到公司后又有一些繁琐的小事情需要办，直到今天才有空开始继续更新手记，让大家久等了，对不起。

今年我会有很多生活上的计划安排，其中比较重要的是我会换一份工作，并且带我的夫人去旅游，业余可能还会参与一个国内Athena项目的二次开发。精力的分配或许会比较饱和，可能会导致手记的更新频度无法像之前承诺一样做到绝对每日一更，不过如果时间和精力允许的话，我会努力做到的！

希望大家羊年万事如意，新的一年可以学到更多的技术，谢谢。
