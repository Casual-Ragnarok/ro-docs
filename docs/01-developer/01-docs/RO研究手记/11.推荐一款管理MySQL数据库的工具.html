推荐一款管理MySQL数据库的工具

2015年1月3日2015年3月15日 Sola丶小克 未分类

前面已经通过《安装MySQL的步骤以及指导说明》向大家介绍如何安装MySQL，安装完成后我们需要通过一些工具，来管理MySQL里面的数据库。

本期推荐的工具为SQLyog Enterprise，下载地址目前放在百度云：http://pan.baidu.com/s/1eQgWBZ0 下面会向大家介绍具体使用办法。

我提供给大家下载的是一个绿色软件，运行里面的SQLyogEnt.exe就可以了。

第一次运行会提示需要注册，请填写压缩包中sn.txt所记录的注册码。

注册成功后，会弹出一个更新提示窗口，由于这个版本已经满足我们的大多数需求了，而且新版本可能也需要其他破解手段，比较麻烦。所以这里建立大家勾选“不再提醒我”…

最终，会看到一个新建连接窗口，我们点击“新建”按钮，可以新建一个链接，连接的名称可以随意命名（你以后可以建立很多链接，链接到一些远程机器的MySQL，在我们这个例子里面，只需要链接本机的MySQL即可）：

然后按照之前安装MySQL设置的root密码和端口，填写到连接界面，并点击“测试连接”按钮：

如果你填写的信息没问题，能够连接上本机的MySQL的话，会看到图下图所示的提示：

成功后点击“连接”按钮，会弹出一个提示界面，问你是否保存这个设置，推荐大家进行保存，下次连接就不用填写上面的信息了：

成功建立链接后，会看到图下图所示的界面，左侧的信息就是本机MySQL中现有的数据库，默认情况下会有4个…分别是information_schema、mysql、performance_schema和test
