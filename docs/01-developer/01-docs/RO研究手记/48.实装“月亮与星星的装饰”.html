实装“月亮与星星的装饰”

2015年1月31日2015年3月15日 Sola丶小克 未分类

暗夜天同学特别希望能在游戏里面把这个头饰给实装了~ 所以本章就来讲一下如何来在游戏中实装”月亮与星星的装饰”。不过实际上不难，因为这个物品的DB信息在复兴后里面是rAthena自带的。

如果你编译的是复兴后的rathena，那么直接在item_db里面可以找到物品ID为18742的一个道具，这个道具就是官服”月亮与星星的装饰”的物品ID了。如果你编译的是复兴前的rathena，那么请去复兴后的item_db里面，将这个编号为18742的物品DB信息复制到复兴前的item_db里面。然后将Class字段从63改成7即可。

复兴后使用的item_db信息（请别直接复制，有可能2个逗号会被博客系统自动合并成1个特殊的逗号，请仔细观察）：

18742,C_MoonStar_Accessory,月亮与星星的装饰,4,20,,0,,0,,0,0xFFFFFFFF,63,2,2048,,0,0,881,{},{ sc_start SC_MOONSTAR,-1,0; },{ sc_end SC_MOONSTAR; }

复兴前使用的item_db信息（请别直接复制，有可能2个逗号会被博客系统自动合并成1个特殊的逗号，请仔细观察）：

18742,C_MoonStar_Accessory,月亮与星星的装饰,4,20,,0,,0,,0,0xFFFFFFFF,7,2,2048,,0,0,881,{},{ sc_start SC_MOONSTAR,-1,0; },{ sc_end SC_MOONSTAR; }

这个时装是比较特殊的，她的效果并不是通过头饰的那种图档文件来实现的，它的效果是一个名为SC_MOONSTAR的特殊状态（所谓状态就是和大声呐喊、霸邪之阵一样的Buff）。

所以大家可以看到在穿戴装备的时候，执行了sc_start SC_MOONSTAR,-1,0;来启动这个外观效果，当脱下装备的时候执行了sc_end SC_MOONSTAR;来关闭这个外观效果……

只要你使用的是2013的客户端的话，应该默认是支持这个状态的，如果不确定的话，可以使用grfmaker2搜索一下data.grf看看有没有moonstar.str和moonstar.ezv文件，有的话就说明至少图档是OK的：

最后只需要完善一下iteminfo信息，就可以了~ 这个物品信息如果你手里持有三转的iteminfo文件的话，应该可以很轻松查到（比如从昆仑RO的客户端里面提取的iteminfo文件里面就可以查到这个物品信息，虽然不完整，但是简单整理一下就可以用了~），下面是从昆仑RO的iteminfo提取到的信息：

大家可以发现，上面iteminfo信息的几个问题，比如：

  1. 没有已经鉴定的物品名称
  2. 没有已经鉴定的图档名称
  3. ClassNum和item_db里面的view对不上
这几个问题中，第一个问题自己手动打上中文名就行，第二个问题要得到答案手续比较复杂，我是从韩服RO的iteminfo信息里面，提取到这个物品的图档名称的，第三个问题也很简单，只需要将ClassNum改成item_db里面指定的881即可，最后的iteminfo信息如下：
