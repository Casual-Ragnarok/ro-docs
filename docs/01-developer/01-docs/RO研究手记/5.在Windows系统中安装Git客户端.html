在Windows系统中安装Git客户端

2014年12月29日 Sola丶小克 未分类

之前我们曾经说过，rAthena的代码托管在Github上，这时候可能会有人打开rAthena在Github上的主页，然后一头雾水不知道应该如何将代码完整的获取下来…

本文主要介绍如何在Windows系统下安装一个Git客户端，方便自己以后获取rAthena的最新代码。

首先你需要安装最新的Git for Windows，主页地址为：http://msysgit.github.io/ 上去后直接点击Download就可以了，下载后请安装上，安装过程中有不少选项，请参考《安装Git for Windows的步骤》。

当你安装完成Git for Windows后，你就可以在cmd里面使用git指令了。但是每次用指令都很麻烦，都要去记住，实在是太累了。

别着急……我们隆重介绍一下TortoiseGit，这是一款很好用的Git客户端，有了它就可以和操作SVN一样，动动鼠标就可以从Github上把代码同步下来了。

它的下载地址是：https://code.google.com/p/tortoisegit/wiki/Download  在下载页面中请根据你的系统类型（32位或64位），下载正确的版本。如果你是中国用户的话，在安装完毕后还可以下载语言包（Language Packs），语言包就在下载页面的底部，如下图所示。

如果你无法访问上面的下载页面，那可能是因为谷歌被封锁导致的，请用迅雷直接下载以下链接，应该可以借助迅雷的解决死链功能来成功下载：

32bit版：点此下载

64bit版：点此下载

32bit专用简体中文语言包：点此下载

64bit专用简体中文语言包：点此下载

如果还是不行，请考虑使用VPN来突破谷歌封锁…

当你把上面这些东西都安装完毕后，请在桌面空白处单击鼠标右键，选择TortoiseGit→Settings，把Language改成中文，然后确定。

至此，你已经成功在Windows中安装了一个Git客户端，以后取代码也容易了。
