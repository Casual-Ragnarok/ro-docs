爱玩仙境 – 物品信息转换器

2015年3月27日 Sola丶小克 原创工具

我们知道老版本的RO客户端使用的data目录下的num2xxx和idnum2xxx等多个txt文件来保存物品的图档名称以及物品描述，但在2013年之后的新版RO客户端中，则使用System目录下的iteminfo.lub文件来保存物品的信息。

若你手里有老版本的num2xxx系列的物品信息文件，可以使用本工具来将其转换成iteminfo格式的文件供新客户端使用。转换后的文件后缀是lua格式的，请根据需要将其编译成lub或者是通过直接修改文件后缀名的方式来使其变成lub格式的文件，然后放到游戏目录下的System文件夹中即可。

产品名称：爱玩仙境 – 物品信息转换器

最新版本：1.0.0.5

更新日期：2015年03月26日

作用说明：此工具可以将以前老格式的道具描述文件，转换成新版客户端所需的iteminfo.lua文件。

下载地址：本地下载

安装说明

若您使用的是Win7及更新的系统，那么可以直接运行本软件。

若您使用的是WinXP或Win2003系统，那么使用本软件之前，您的电脑需 要先安装 Microsoft .NET Framework 4 Client Profile 否则的话 运行可能会出错。

您可以到微软网站上下载这个 Framework 组件： http://download.microsoft.com/download/5/6/2/562A10F9-C9F4-4313-A044-9C94E0A8FAC8/dotNetFx40_Client_x86_x64.exe

使用说明

1、将你要处理的num2xxx、idnum2xxx以及itemslotcounttable.txt等文件解压到某个目录

2、将你的item_db.txt也放到这个目录里面

3、然后打开工具，选择右上角的“浏览”来选择刚刚你解压出来的目录

4、点击“开始转换”选择iteminfo.lua文件的保存路径即可。
